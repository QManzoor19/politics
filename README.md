# Politics

A self-paced introductory politics course built as a single-page web app. Same shell as `psychlearn`/`econlearn` — vanilla JS, PWA, dark purple theme.

## Curriculum (9 units)

1. What Is Politics?
2. Political Thinkers
3. Ideologies
4. Forms of Government
5. US Government
6. Political Participation
7. Comparative Politics
8. International Politics
9. Current Debates

Lesson content, quiz questions, glossary, and videos are stubbed pending build-out.

## Run locally

Open `index.html` in any modern browser. No build step.

## AI features (optional)

Add an Anthropic API key in **Settings (⚙️)** to enable the AI tutor.

## Stack

Vanilla HTML, CSS, JavaScript. No frameworks, no build tools. PWA-ready (`sw.js`, `manifest.json`).
